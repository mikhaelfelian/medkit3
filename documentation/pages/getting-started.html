Getting Started

Welcome to NUSANTARA HMVC! This guide will help you get up and running quickly.

System Requirements

  • PHP 7.4 or higher
  • MySQL 5.7 or higher
  • Apache/Nginx web server
  • mod_rewrite enabled
  • PDO PHP Extension
  • JSON PHP Extension
  • OpenSSL PHP Extension
  • Mbstring PHP Extension

Installation

1. Download Framework

# Clone repository
git clone https://github.com/yourusername/nusantara-hmvc.git

# Navigate to project directory
cd nusantara-hmvc

2. Configure Web Server

For Apache (.htaccess):

RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php/$1 [L]

For Nginx:

location / {
    try_files $uri $uri/ /index.php?$query_string;
}

3. Configure Application

// config/config.php
$config = [
    'app_name' => 'Your Application',
    'base_url' => 'http://localhost/your-app',
    'timezone' => 'Asia/Jakarta',
    'charset' => 'UTF-8',
    'debug_mode' => true,
    'encryption_key' => 'your-32-char-key',
    'session_handler' => 'files', // files, database
    'maintenance_mode' => false
];

// Database configuration
$db_config = [
    'development' => [
        'hostname' => 'localhost',
        'username' => 'root',
        'password' => '',
        'database' => 'your_database',
        'charset' => 'utf8mb4',
        'collation' => 'utf8mb4_unicode_ci'
    ]
];

4. Set Directory Permissions

# Set write permissions
chmod -R 755 public/
chmod -R 755 app/logs/
chmod -R 755 app/cache/

Creating Your First Page

1. Create Controller

// app/controllers/HomeController.php
class HomeController extends BaseController {
    public function __construct() {
        parent::__construct();
    }
    
    public function index() {
        return $this->view('home/index', [
            'title' => 'Welcome to NUSANTARA HMVC'
        ]);
    }
}

2. Create View


<div class="container">
    <h1><?= $title ?></h1>
    <p>Your first page with NUSANTARA HMVC!</p>
</div>

3. Create Model (Optional)

// app/models/UserModel.php
class UserModel extends BaseModel {
    protected $table = 'users';
    protected $primaryKey = 'id';
    protected $fillable = ['name', 'email'];
}

Basic Usage

Routing

URL Pattern: http://your-app/controller/method/param1/param2

Examples:
http://your-app/home            // HomeController->index()
http://your-app/users/show/1    // UsersController->show($id)

Database Operations

// In your controller
$this->loadModel('User');

// Create
$this->model->create([
    'name' => 'John Doe',
    'email' => 'john@example.com'
]);

// Read
$user = $this->model->find(1);

// Update
$this->model->update(1, ['name' => 'Jane Doe']);

// Delete
$this->model->delete(1);
