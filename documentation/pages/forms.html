Forms & Validation

Form Helper

The BaseForm class provides methods for form creation and handling:

  • input() - Create input fields
  • select() - Create select dropdowns
  • textarea() - Create textarea fields
  • csrf() - Add CSRF protection
  • validation - Handle form validation

Example Usage


$form = BaseForm::getInstance();

// Open form
echo $form->open('pasien/store', 'POST');

// Create input
echo $form->input('text', 'nama', '', ['required' => true]);

// Create select
echo $form->select('status', [
    'active' => 'Active',
    'inactive' => 'Inactive'
]);

// Close form
echo $form->close();
